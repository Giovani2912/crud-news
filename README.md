# Como baixar:

# Backend
```

    $ git clone https://github.com/Giovani2912/crud-news.git

    $ cd api

# Caso esteja utilizando o npm

    $ npm install

    $ npm start

# Caso esteja utilizando o yarn

    $ yarn 

    $ yarn start


```
# Frontend
```

    $ git clone https://github.com/Giovani2912/crud-news.git

    $ cd admin

# Caso esteja utilizando o npm

    $ npm install

    $ npm start

# Caso esteja utilizando o yarn

    $ yarn 

    $ yarn start
